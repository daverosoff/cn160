Instead of introducing symbols for NOR, NAND, and XNOR, we can make use of symbols for a unary NOT operator as described in class. We will write \(\neg x\) for \(\text{NOT } x\text{,}\) and we will also use the overbar to negate a boolean expression. So, for example, \(\neg x = \overline{x}\text{,}\) and
\begin{equation*} \overline{x \wedge y} = \neg (x \wedge y) = \text{NOT } (x \wedge y) \end{equation*}
and all three expressions are synonymous.
