In doing Activity 1.2.2, you showed that the NOR operator is a universal gate. This means that any boolean function can be expressed using only NORs. To spell it out, we used NORs to express \(\neg x\text{,}\) \(x \wedge y\text{,}\) \(x \vee y\text{,}\) and \(x \veebar y\text{.}\) The remaining operators, NAND and XNOR, can be expressed in terms of these four. This shows that all six binary boolean operators (that is, those represented in Table 1.2.2–1.2.7 ) are expressible in terms of NOR alone.
