Activity 1.2.2. Universal gates.

(a)

Use a single NOR operator to express \(\neg x\text{.}\)
Hint.
The NOR operator has two inputs, and you only have \(x\text{.}\) What can you do?
Answer.
\(\neg x = x \text{ NOR } x\text{.}\)

(b)

Use the result of Task 1.2.1.a to express \(x \wedge y\) using only the NOR operator. (You can use as many NORs as you need.)

(c)

Use the result of Task 1.2.1.b to express \(x \vee y\) using only the NOR operator. (You can use as many NORs as you need.)

(d)

Use NOR operators to express \(x \veebar y\text{.}\)
Hint.
Use the results of Task 1.2.2.b and Task 1.2.2.c.
